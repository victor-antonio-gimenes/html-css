lista ordenada

As listas ordenadas são feitas utilizado a tags < ol > para iniciar uma lista ordenada, com seu items dentro dela < li >. A lista ordenada utiliza uma ordem numerica de items adicionados

    Lista de compras
  1. maça
  2. banana
  3. uva

Lista não ordenada

Como o titulo sugere, ela é organizada de forman não ordenada com a tags < ul > que inicia a lista e seus items < li >

    Lista de desejos
  • escalar a montanha everest
  • chutar gideao
  • capotar o corsa

Misturando listas

Utilizado um pouco de criatividade voce pode misturar as duas listas

    lista de analise
  1. computador
      hardware:
    • Nvidia GT 730
    • CPU I7-2600
  2. carro

Lista de definição

A lista de definição ela serve para definir ou explicar uma entidade, com a tags < dl > que inicia a lista, o < dt > que adiciona um titulo ou entidade referente e < dd > para dar uma definição ou descrição a entidade
